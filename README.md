# Portable-MSVC-compiler-linker-Windows-SDK

## Setup
- Run the msvc.py file (folders "download" and "msvc" will be created)
- Delete the "download" folder as not needed
- Put the "msvc" folder wherever you want it to be
- Copy the "permanent_setupx64.bat" and "remove_permanent_setupx64.bat" into "msvc" folder
- Run "permanent_setupx64.bat" (to set all the environment variables)

## Removal
- Run "remove_permanent_setupx64.bat" (to remove all the environment variables)
- Delete the "msvc" folder
