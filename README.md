# Portable-MSVC-compiler-linker--Windows-SDK <hr>

## Setup
step 1 :- Run the msvc.py file (folders "download" and "msvc" will be created)<br>
step 2 :- Delete the "download" folder as not needed<br>
step 3 :- Put the "msvc" folder wherever you want it to be<br>
step 4 :- Copy the "permanent_setupx64.bat" and "remove_permanent_setupx64.bat" into "msvc" folder<br>
step 5 :- Run "permanent_setupx64.bat" (to set all the environment variables)<br>

## Remove
step 1 :- Run "remove_permanent_setupx64.bat" (to remove all the environment variables)<br>
step 2 :- Delete the "msvc" folder<br>
