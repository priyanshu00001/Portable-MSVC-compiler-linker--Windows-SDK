===Setup===
step 1 :- Run the msvc.py file (folders "download" and "msvc" will be created)
step 2 :- Delete the "download" folder as not needed
step 3 :- Put the "msvc" folder wherever you want it to be
step 4 :- Copy the "permanent_setupx64.bat" and "remove_permanent_setupx64.bat" into "msvc" folder
step 5 :- Run "permanent_setupx64.bat" (to set all the environment variables)

===Remove===
step 1 :- Run "remove_permanent_setupx64.bat" (to remove all the environment variables)
step 2 :- Delete the "msvc" folder
